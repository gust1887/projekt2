Hvordan fungerer Bit Chat?

Bit Chat er udviklet som en enkel og sikker kommunikationsplatform for både værter og deltagere. Systemet samler al dialog ét sted, så alle involverede hurtigt kan komme i kontakt med hinanden før, under og efter et event.

Sådan fungerer Bit Chat:

1. Opret en konto
Brugere registrerer sig som vært eller deltager med navn, e-mail og adgangskode.

2. Log ind og få adgang til dine samtaler
Efter login får du overblik over dine igangværende samtaler samt mulighed for at starte nye.

3. Start eller deltag i en samtale
Værten kan oprette chats med deltagere, og begge parter kan skrive til hinanden i realtid.

4. Sikker beskedudveksling
Al kommunikation sendes via HTTPS og håndteres sikkert på serveren. Adgangskoder gemmes krypteret, og brugerne kan nulstille dem via e-mail.

Kort sagt: Bit Chat gør det nemt for værter og deltagere at holde styr på dialogen fra start til slut.
